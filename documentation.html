Game PC Builder

Architecture & API Documentation

1. Application Overview

The Game PC Builder is a web application that helps users find optimal PC builds for popular games based on their budget. The application provides component recommendations, pricing in pounds, and direct purchase links.

Technology Stack

  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Styling: Custom CSS with CSS Variables
  • Build Tools: None (Vanilla JS)
  • Deployment: Static file hosting

Key Components

  • User Interface: Responsive design for all device sizes
  • Component Database: Local JSON structure with pricing in GBP
  • Build Engine: Algorithm that selects optimal components based on game requirements and budget
  • Game Optimizations: Specific performance profiles for each supported game

3. Data Flow

sequenceDiagram participant User participant UI as User Interface participant BE as Build Engine participant DB as Component DB User->>UI: Selects Game & Budget UI->>BE: Game Type, Budget BE->>DB: Request Components DB-->>BE: Return Component Data BE->>BE: Calculate Optimal Build BE-->>UI: Recommended Build UI->>User: Display Results

Process Flow

  1. User selects a game and sets their budget in pounds
  2. Application loads the appropriate optimization profile for the selected game
  3. Build engine selects components based on budget and game requirements
  4. Results are displayed with component details and purchase links

Build Generation Endpoint

GET /api/build?game=:game&budget=:budget

Generates a PC build based on the specified game and budget.

Parameters

  • game (required): Game identifier (e.g., 'fortnite', 'lol')
  • budget (required): Budget in pounds (e.g., 800)

Response

{
  "success": true,
  "build": {
    "cpu": { "name": "...", "price": 159, "url": "..." },
    "gpu": { "name": "...", "price": 399, "url": "..." },
    // ... other components
  },
  "total": 1200,
  "currency": "GBP"
}
